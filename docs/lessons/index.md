# Lessons

Individual lessons covering specific cloud engineering topics with hands-on exercises.

## Lesson Structure

Every lesson follows a consistent structure:

1. **Learning Objectives** — What you'll be able to do
2. **Prerequisites** — What you need to know
3. **Core Content** — The main material with diagrams and code
4. **Hands-On Exercise** — Practice what you learned
5. **Key Takeaways** — Summary of main points
6. **Check Your Understanding** — Self-assessment questions
