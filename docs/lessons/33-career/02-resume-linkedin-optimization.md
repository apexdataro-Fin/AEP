# تحسين السيرة الذاتية و LinkedIn

> "سيرتك الذاتية لديها 6 ثوانٍ لتثير الإعجاب. اجعل كل كلمة تحسب."

## 🎯 أهداف التعلم

- كتابة سيرة ذاتية تجتاز ATS
- تحسين LinkedIn Profile
- Keywords المناسبة لـ Cloud Engineer
- أمثلة عملية

## ⏱️ الوقت المقدر: 30 دقيقة | المستوى: Junior

---

## 🏗️ قواعد السيرة الذاتية

### 1. ATS Optimization

ATS (Applicant Tracking System) يبحث عن keywords:

```
✅ صحيح: "نشرت Kubernetes cluster على Azure AKS لـ 50 خدمة"
❌ خاطئ: "عملت على مشاريع cloud"

✅ صحيح: "خفضت فاتورة Azure بنسبة 40% عبر right-sizing و Reserved Instances"
❌ خاطئ: "وفرت فلوس للشركة"
```

### 2. هيكل bullet points

```
[فعل] + [ماذا] + [كيف] + [نتيجة/رقم]

مثال:
"أتمتة [فعل] نشر البنية التحتية [ماذا] باستخدام Terraform و CI/CD [كيف]
مما قلل وقت النشر من 3 أيام إلى 15 دقيقة [نتيجة]"
```

### LinkedIn Optimization

| العنصر | أفضل ممارسة |
|--------|-----------|
| **الصورة** | احترافية، خلفية نظيفة |
| **العنوان** | "Cloud & DevOps Engineer | Azure | Kubernetes | Terraform" |
| **About** | قصة قصيرة: من أنت + ماذا تفعل + ماذا تريد |
| **المهارات** | Azure, Kubernetes, Terraform, Docker, CI/CD, Python |
| **التوصيات** | اطلب من زملاء ومديرين سابقين |

## 🛠️ تدريب

1. راجع سيرتك الذاتية الحالية واحذف أي bullet point بدون رقم
2. أضف 3 keywords من وصف وظيفة أحلامك
3. حسن LinkedIn headline
