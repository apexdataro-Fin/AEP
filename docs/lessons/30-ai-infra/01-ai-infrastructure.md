# بنية الذكاء الاصطناعي التحتية

> **"نماذج AI تحتاج بنية تحتية خاصة. ليست مجرد خوادم — تحتاج GPUs، وذاكرة عالية، وشبكات سريعة."**

## خيارات GPU على Azure

| الخدمة | المعالج | VRAM | الاستخدام | التكلفة/ساعة |
|---|---|---|---|---|
| **NCas T4 v3** | Tesla T4 | ١٦GB | استدلال خفيف | ~$٠.٥٠ |
| **NC A100 v4** | A100 | ٨٠GB | تدريب، استدلال ثقيل | ~$٣.٥٠ |
| **ND H100 v5** | H100 | ٨٠GB | تدريب نماذج ضخمة | ~$٥-٨ |
| **Azure ML Compute** | تلقائي | متغير | تدريب مُدار | حسب الاستخدام |

## استراتيجيات النشر

| الاستراتيجية | زمن الاستجابة | التكلفة | الأنسب لـ |
|---|---|---|---|
| **Real-time endpoint** | < ١٠٠ms | $$$ | تطبيقات المستخدم |
| **Batch inference** | دقائق-ساعات | $ | معالجة ليلية |
| **Serverless (Azure OpenAI)** | < ١s | $/token | تطبيقات ChatGPT-like |

## تحسين التكلفة

```python
# مثال: تكلفة GPT-4 يومياً
# متوسط: ١٠٠٠ سؤال يومياً × ٢٠٠٠ token للسؤال = ٢ مليون token
# السعر: $0.03/1K input + $0.06/1K output
# التكلفة اليومية: ~$٩٠
# التكلفة الشهرية: ~$٢٧٠٠

# تحسينات:
# ١. استخدم GPT-4o-mini للأسئلة البسيطة (أرخص ١٠x)
# ٢. خزّن الإجابات المتكررة (caching)
# ٣. حدد عدد tokens في الـ prompt
```
