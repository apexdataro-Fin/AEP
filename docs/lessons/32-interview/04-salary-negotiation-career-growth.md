# التفاوض على الراتب والنمو الوظيفي

> "الراتب الذي تبدأ به يحدد مسارك لسنوات. تفاوض بذكاء."

## 🎯 أهداف التعلم

- متى وكيف تتفاوض
- أبحاث السوق (Glassdoor, Levels.fyi)
- الثقة في التفاوض
- التخطيط للترقية

## ⏱️ الوقت المقدر: 30 دقيقة | المستوى: Junior

---

## 🏗️ استراتيجية التفاوض

### 1. ابحث أولاً

```python
# متوسط رواتب Cloud Engineer (السعودية 2026)
salaries = {
    "Junior Cloud Engineer": "120,000 - 180,000 SAR",
    "Cloud Engineer": "180,000 - 280,000 SAR",
    "Senior Cloud Engineer": "280,000 - 400,000 SAR",
    "Cloud Architect": "400,000 - 600,000 SAR",
    "Principal Architect": "600,000+ SAR"
}
```

### 2. اعرف قيمتك

- Azure certifications (AZ-104, AZ-400)
- Kubernetes (CKA, CKAD)
- Projects منشورة على GitHub
- خبرة عملية (سنوات + scale)

### 3. توقيت التفاوض

- ✅ عند استلام عرض عمل
- ✅ عند الترقية
- ✅ بعد مشروع ناجح كبير
- ✅ في المراجعة السنوية
- ❌ في منتصف أزمة شركة

### 4. لغة التفاوض

```
بدلاً من: "أريد زيادة"
قل: "بناءً على إنجازاتي في [مشروع X] حيث وفرت [Y]،
أعتقد أن راتبي يجب أن يعكس هذه القيمة. ما رأيك؟"
```

### خطة 5 سنوات

| السنة | الهدف | الشهادة | الراتب المستهدف |
|-------|-------|---------|---------------|
| 1 | Junior Cloud Engineer | AZ-900 | 120K |
| 2 | Cloud Engineer | AZ-104 | 180K |
| 3 | Senior | AZ-400 + CKA | 280K |
| 4 | Architect | AZ-305 | 400K |
| 5 | Principal / Lead | - | 500K+ |

## 🏛️ CloudNova: مفاوضة غيرت مستقبل مهندس

**فيصل** Junior Cloud Engineer. بعد سنتين في CloudNova، تلقى عرضاً من شركة منافسة: راتب 180,000 SAR.

راتبه الحالي: 120,000 SAR. المدير لا يعرف.

**الخطأ الشائع:** يذهب للمدير ويقول: "لدي عرض. زد راتبي أو سأرحل."

**الصحيح:**

1. جمع أدلة الإنجاز:
```
- أتمتة نشر 30 تطبيقاً (وفر 200 ساعة/شهر)
- خفض فاتورة Azure 40% (وفر 180,000 SAR سنوياً)
- قاد ترحيل 5 خدمات إلى AKS (zero downtime)
- حصل على AZ-104 + AZ-400
```

2. بحث السوق:
- Glassdoor: متوسط Senior Cloud Engineer في الرياض = 240,000-320,000 SAR
- Levels.fyi: المتوسط في الشركات التقنية = 280,000 SAR
- معارفه في LinkedIn: 3 تأكيدات على النطاق

3. المحادثة:
> "أريد مناقشة تطور مساري. خلال السنتين الماضيتين، حققت [X, Y, Z]. حسب أبحاث السوق، متوسط Senior Cloud Engineer هو 280,000. أعتقد أن مهاراتي ومساهماتي تبرر هذا المستوى. ما رأيك؟"

4. النتيجة:
- المدير: "أتفق. سأرفع طلب ترقية إلى Senior مع تعديل الراتب."
- بعد 3 أسابيع: Senior Cloud Engineer، راتب 260,000 SAR (+117%!)

## 🎨 استراتيجيات التفاوض

### 5 أخطاء قاتلة في التفاوض

1. **التهديد:** "زد راتبي أو أرحل" → يحرق الجسور
2. **بدون أرقام:** "أعتقد أني أستحق زيادة" → لا دليل
3. **مقارنة خاطئة:** "فلان راتبه أكثر" → غير مهني
4. **توقيت سيء:** أثناء أزمة مالية للشركة
5. **قبول أول عرض:** دائماً تفاوض (باحترام)

### متى تطلب زيادة؟

| التوقيت | فرصة النجاح | ملاحظات |
|--------|-----------|--------|
| ✅ بعد مشروع ناجح كبير | 80% | أفضل توقيت |
| ✅ المراجعة السنوية | 70% | جهز أدلتك مسبقاً |
| ✅ بعد شهادة مهمة | 60% | AZ-104, CKA |
| ✅ عند استلام عرض خارجي | 50% | بحذر، لا تهديد |
| ❌ في أزمة شركة | 10% | انتظر |
| ❌ قبل إثبات نفسك | 5% | ابنِ سجلك أولاً |

### خطة 5 سنوات — من Junior إلى Principal

| السنة | المسمى | الشهادة | الراتب (SAR) | المهارات |
|-------|--------|---------|-------------|---------|
| 1 | Junior Cloud Engineer | AZ-900 | 120K-150K | Azure basics, Linux |
| 2 | Cloud Engineer | AZ-104 | 180K-220K | Terraform, Docker, CI/CD |
| 3 | Senior Cloud Engineer | AZ-400 + CKA | 260K-320K | Kubernetes, Design |
| 4 | Cloud Architect | AZ-305 | 350K-450K | Architecture, Multi-cloud |
| 5 | Principal/Lead | - | 500K+ | Leadership, Strategy |

## 🛠️ تدريبات عملية

### تمرين 1: جهز ملف إنجازاتك
```markdown
# ملف الإنجازات الشخصي

## المشاريع الرئيسية
1. **ترحيل منصة payment إلى AKS**
   - وفر: 40% تكلفة + 90% وقت نشر
   - التقنيات: AKS, Terraform, GitHub Actions
   - الفريق: قُدت 3 مهندسين

2. **نظام مراقبة شامل**
   - غطى: 200+ خدمة عبر 3 بيئات
   - التقنيات: Prometheus, Grafana, Azure Monitor

## الشهادات
- AZ-104 (2025)
- CKA (2026)
- AZ-400 (2026)

## أرقام مهمة
- وفرت للشركة: ~500,000 SAR سنوياً
- أتمتة: 80% من عمليات النشر
- قيادة: 3 مشاريع كبيرة
```

### تمرين 2: تدرب على المحادثة
```
تدرب مع صديق أو أمام المرآة:

1. "أريد مناقشة مساري المهني..."
2. "خلال السنة الماضية، حققت..."
3. "حسب أبحاث السوق..."
4. "ما رأيك في خطة لتطوير مساري؟"

المفتاح: ثقة + أدلة + احترام
```

### تحدي: خطة 5 سنوات مفصلة
```
اكتب خطتك لـ 5 سنوات:
- كل سنة: target role + certification + salary
- كل سنة: 3 skills جديدة
- كل سنة: مشروع كبير واحد
- كل سنة: هدف network (mentor, conference, community)
```

## 📝 تقييم

### ✅ Knowledge Checks
1. ما أفضل توقيت لطلب زيادة؟
2. كم يجب أن تطلب زيادة (نسبة مئوية)؟
3. كيف تبحث عن متوسط الرواتب؟
4. ماذا تفعل إذا رُفض طلبك؟
5. ما الفرق بين lateral move و promotion؟

### 🧠 Quiz
**س1:** أفضل طريقة لطلب زيادة:
- أ) "أحتاج فلوس أكثر"
- ب) عرض إنجازات + أبحاث سوق + طلب مهني ✅
- ج) تهديد بالاستقالة
- د) مقارنة بالزملاء

**س2:** راتب Senior Cloud Engineer في السعودية:
- أ) 50K
- ب) 120K
- ج) 260K-320K ✅
- د) مليون

**س3:** بعد رفض طلب الزيادة:
- أ) استقيل فوراً
- ب) اسأل: "ما المطلوب للترقية؟" وخطة زمنية ✅
- ج) أصرخ
- د) لا تفعل شيئاً

### 🗣️ Active Recall
1. صف خطوات التفاوض الخمس
2. ارسم خطة 5 سنوات من Junior إلى Principal
3. كيف تحسب قيمتك السوقية؟
4. متى تترك الشركة لمطاردة راتب أعلى؟

### 🎓 Feynman Exercise
> اشرح التفاوض لصديق: "مثل بيع سيارة. لا تقل 'أريد أكثر'. قل 'سعر السوق 50,000. سيارتي عليها إضافات X, Y, Z. صيانتها كاملة.' المشتري يرى القيمة، لا المطالبة."

### 🃏 بطاقات تعلم
| السؤال | الإجابة |
|--------|---------|
| أفضل توقيت للزيادة؟ | بعد مشروع ناجح كبير |
| كم زيادة أطلب؟ | 15-30% بناءً على السوق |
| أين أبحث عن الرواتب؟ | Glassdoor, Levels.fyi, LinkedIn |
| ماذا لو رُفضت؟ | اسأل عن خطة واضحة للترقية |
| أهم نصيحة؟ | ابنِ قضيتك بالأرقام، لا بالمشاعر |

## 🎤 أسئلة المقابلة

**س1:** "كم راتبك المتوقع؟"
> "بناءً على أبحاثي للسوق ومهاراتي في [X, Y, Z]، أتوقع نطاق 260,000-300,000 SAR. لكني مرن وأهم شيء الفرصة المناسبة."

**س2:** "لماذا تترك عملك الحالي؟"
> "أبحث عن تحدٍ أكبر. في منصبي الحالي، وصلت لسقف النمو التقني. أريد بيئة مثل [اسم الشركة] حيث يمكنني [هدف محدد]."

**س3:** "أين ترى نفسك بعد 5 سنوات؟"
> "Cloud Architect أو Principal Engineer. لدي خطة: السنة 1-2 تعمق في [تقنية]، سنة 3-4 قيادة مشاريع أكبر، سنة 5 مساهمة في استراتيجية المنصة."
