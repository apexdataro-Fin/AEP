# التفاوض على الراتب والنمو الوظيفي

> "الراتب الذي تبدأ به يحدد مسارك لسنوات. تفاوض بذكاء."

## 🎯 أهداف التعلم

- متى وكيف تتفاوض
- أبحاث السوق (Glassdoor, Levels.fyi)
- الثقة في التفاوض
- التخطيط للترقية

## ⏱️ الوقت المقدر: 30 دقيقة | المستوى: Junior

---

## 🏗️ استراتيجية التفاوض

### 1. ابحث أولاً

```python
# متوسط رواتب Cloud Engineer (السعودية 2026)
salaries = {
    "Junior Cloud Engineer": "120,000 - 180,000 SAR",
    "Cloud Engineer": "180,000 - 280,000 SAR",
    "Senior Cloud Engineer": "280,000 - 400,000 SAR",
    "Cloud Architect": "400,000 - 600,000 SAR",
    "Principal Architect": "600,000+ SAR"
}
```

### 2. اعرف قيمتك

- Azure certifications (AZ-104, AZ-400)
- Kubernetes (CKA, CKAD)
- Projects منشورة على GitHub
- خبرة عملية (سنوات + scale)

### 3. توقيت التفاوض

- ✅ عند استلام عرض عمل
- ✅ عند الترقية
- ✅ بعد مشروع ناجح كبير
- ✅ في المراجعة السنوية
- ❌ في منتصف أزمة شركة

### 4. لغة التفاوض

```
بدلاً من: "أريد زيادة"
قل: "بناءً على إنجازاتي في [مشروع X] حيث وفرت [Y]،
أعتقد أن راتبي يجب أن يعكس هذه القيمة. ما رأيك؟"
```
