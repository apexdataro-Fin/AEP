# Changelog

All notable changes to this project are documented here.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] — 2026-07-15

### Added

- **Foundation**
  - Docusaurus 3.10 project scaffold
  - TypeScript configuration
  - Complete directory structure
  - Professional dark/light theme (sky-blue/slate palette)

- **Documentation**
  - Architecture: overview, system design, data flow, tech stack, security model
  - Development: getting started, environment setup, project structure, workflows, testing, deployment
  - Standards: style guide, naming conventions, code quality, documentation standards, content standards, metadata guide
  - Guides: Mermaid guide, AI integration, contributing, knowledge graph
  - Reference: glossary, roadmap, changelog, quality standards, API reference

- **Platform Features**
  - PWA support with service worker and offline mode
  - Local full-text search
  - Mermaid diagram rendering with light/dark themes
  - Responsive design (mobile, tablet, desktop)
  - Print styles

- **Infrastructure**
  - GitHub Pages deployment via GitHub Actions
  - Content validation scripts
  - Metadata schemas
  - Content templates
  - Prettier and ESLint configuration
